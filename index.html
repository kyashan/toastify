Toastify

Toast messages for angularjs-based apps

https://github.com/kyashan

Toastify public methods

Toastify returns a set of public methods you can use to modify the toast instance you created with create method (the first in the following list):

toastify.create(params) //Create a new toast instance

toastify.done(toast) //Apply to toast done options specified in create method

toastify.closeToast(toast) //Close the toast instance passed as parameter

toastify.closeAllToasts() //Close all toasts instances

Toastify default parameters to pass inside toastify.create method

{
  toastCustomClass: null,
  appendTo: 'body', //Id of html tag where to append toasts
  closeByClick: false,
  showClose: true,
  autoclose: true,
  timeout: 4000,
  message : '',
  callback: null, //Callback when toast is closed
  onclick: null, //Methods to call when click on entire toast is performed
  template: null,
  controller: null,
  gravity: 'top', //Possible values are 'top' or 'bottom'
  doneMessage: '',
  doneClass: 'done', //Css class to apply on done method
  doneAutoclose: true,
}

Show a simple autoclosing toast

toastify.create({
  timeout: 2000,
  message: 'Autoclosing toast',
});

Show a simple error toast with callback

function alertCallback({
  alert('Toast closure callback')
});

toastify.create({
  toastCustomClass: 'error',
  autoclose: false,
  message: 'Error message',
  callback: alertCallback,
});

Show a toast with a success message

/*CSS*/
.onDone {
  background-color: #0091FF;
  border-color: #0270c3;
}

//Javascript
var toast = toastify.create({
  autoclose: false, // if autoclose set to false, timeout refers to done duration
  timeout: 2000,
  showClose: false,
  message: 'Waiting for done...',
  doneClass: 'onDone',
  doneMessage: 'Done!'
});

setTimeout(function() {
  tostify.done(toast);
}, 2000);

Show a toast with a custom template

toastify.create({
  timeout: 4000,
  template: <i style="color: rgb(255, 239, 153)">This is a template</i>
});

Show permanent toast at bottom of the page

toastify.create({
  showClose: false,
  autoClose: false,
  gravity: 'bottom',
  template: <p style="padding: 0 20px 0 20px">Click <u ng-click="closeThisToast()" style="cursor: pointer; color: rgb(255, 239, 153)">here</u> to close the toast</p>
});

Toast private methods

Each toast created by create method comes with a set of methods attached to its scope.

closeThisToast() //Close the toast instance. See example at previous point

getToast() //Return toast object

closeAllToasts() //Close all toasts instances

Toastify events

Anytime a new toast has been created, is removing or has been removed, the following messages are broadcasted all over the web app

//On new toast creation
$scope.$on('toastify.toast-created', function(event, args){
//args.toast to get the toast instance
});


//When toast is removed (not considering animation duration)
$scope.$on('toastify.toast-removing', function(event, args){
//args.toast to get the toast instance
});


//When toast is removed and animation is concluded
$scope.$on('toastify.toast-removed', function(event, args){
//args.toast to get the toast instance
});
